# judge_talk
動画のtalking判定  

## install
```
pip install ffmpeg
```

## requirements
 - prepare google speech api                       
 https://qiita.com/knyrc/items/7aab521edfc9bfb06625  

## Flow
 - 動画を音声に変換  
https://www.kinacon-blog.work/entry/2018/10/23/210211  
 - Google認識で、動画音声を読み込ませて100文字以上ならtalking
